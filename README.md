# HomeoMaster

An offline-first homeopathic repertory and materia medica app built with React Native (Expo).

## Features
- **Comprehensive Database**: Over 600+ remedies with keynotes, modalities, and clinical indications.
- **Body-Part Navigation**: Intuitive symptom search by body location.
- **Case Simulator**: Interactive clinical cases to test your prescribing skills.
- **My Repertory**: Select symptoms to see ranked remedy suggestions.
- **Offline Access**: Fully functional without internet connection.
- **Quiz Mode**: Test your knowledge of Materia Medica.

## Setup

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start the App**:
    ```bash
    npx expo start
    ```

## Data Sources
- **Core Polish**: Manually curated Polychrests and Reckeweg combinations.
- **OOREP Import**: Large dataset imported from the Open Homeopathic Repertory Project (OOREP).

## Tech Stack
- React Native (Expo)
- TypeScript
- NativeWind (Tailwind CSS)
- Zustand (State Management)
- Async Storage (Persistence)
